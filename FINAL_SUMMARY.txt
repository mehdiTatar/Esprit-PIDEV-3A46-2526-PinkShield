📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: "Column 'nom' not found"
Solution: Run setup_database_FIXED.sql

Problem: "Connection refused"
Solution: Make sure MySQL is running (check Services)

Problem: App shows no products
Solution: Click "Refresh Products" button

Problem: Error at startup
Solution: Check console output, see DATABASE_FIX_GUIDE.txt

Problem: MySQL command not found
Solution: Add MySQL to PATH or use full path:
   C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql -u root
